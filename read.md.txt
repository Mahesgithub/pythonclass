# PDF Chatbot

This project is a conversational PDF chatbot built with FastAPI and OpenAI or open-source LLM APIs.

## Features

- Upload PDF files
- Chat with the content of the PDF using Retrieval-Augmented Generation (RAG)
- Integration with OpenAI / Hugging Face models
